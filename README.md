# dotfiles
Clone this repo into your home folder.

```
$ git clone git@github.com:primalivet/dotfiles.git ~
```

Basically just run the install.sh script to get up and running.

```
$ ~/dotfiles/install.sh
```

If there're aleardy existing dotfiles in your home `~` the script will create a backup folder with those files at `~/dotfiles-backup-YYYYMMDDHHMM`.
